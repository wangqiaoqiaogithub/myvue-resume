# myvue-resume

> A Vue.js project

# 构建安装程序

``` bash
# 安装依赖
npm install

# 服务往往运行在localhost:8080
npm run dev

# 线上打包命令
npm run build

# 建立捆绑和分析报告
npm run build --report
```
